## Part 1 — Expand `LedgerOperationsPanels.test.ts`

### What the component does

`LedgerOperationsPanels` renders a reconciliation form with:
- Account selector, statement date input, statement balance input
- A `reconciliation-note` line when `reconciliationBook.latestSession` is set
- Two summary cards: "Cleared total" and "Difference" — the Difference card gets class `balanced` when `reconciliationBook.difference === 0`, class `warning` otherwise
- A candidate transaction list — each candidate renders as a button showing description, date/payee, amount, and `selected` state (`"Cleared"` / `"Open"`)
- A submit button that is `disabled` when `props.busy !== null` and shows `"Reconciling..."` when `props.busy === "Reconciliation"`
