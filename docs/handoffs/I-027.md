## Context

The register tab row currently uses rounded pill-style wrappers with a left/right reorder buttons per tab, plus an account `<select>` + "Open tab" button for opening new tabs. The goal is proper browser/VS Code-style tabs and a minimal `+` button to the right of the last tab. The account selector dropdown is removed â€” the COA panel (I-028) becomes the navigation mechanism.

When `+` is clicked it opens a new "All accounts" tab and makes it active. The user then clicks an account in the COA panel to load it (I-028 wires that up). The `+` button is always visible regardless of how many tabs are open.
